# Example Recipe

Version: 1.0  
Status: To test

## Summary

This is what a WellEating recipe looks like. Replace it with your own.

## Time

15 minutes

## Ingredients

- One markdown file per recipe
- A folder per category (dinner, fish, salads...)
- Optional Italian sibling file ending in .it.md

## Method

1. Copy this file and rename it.
2. Write your recipe in the same structure.
3. Restart the container, or rebuild if you bake content into the image.
4. Promote the status once you have actually cooked it.
